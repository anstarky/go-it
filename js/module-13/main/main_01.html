Задание 1

Пришло время настроить коммуникацию между фронтендом и бекендом, самую важную часть любого современного приложения. Бекенд отвечает за хранение данных, клиент запрашивает данные и отрисовывает интерфейс. Вся работа с данными проходит через сервер и никак иначе: чтение, создание, удаление, обновление.

  • Удали код отвечающий за сохранение заметок в localStorage, больше тебе это не понадобится.
  • Добавь в проект пакет json-server, скрипт для запуска сервера в package.json и используй данные ниже для db.json. Бекенд локальный, для разработки как раз то что надо.
  • Создай файл src/services/api.js в котором напиши функции для HTTP-запросов на бекенд. Используй эти функции в методах класса Notepad.
